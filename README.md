# 开智正典：心智升级百本经典导读



开智社群集体评选108本经典神作， 以“不收录近作，世界级水准，开启了一个学派或者反响极大，或者保留了某个学术传统”为收录原则，集体导读百本令你脑洞大开、三观尽改、代表21世纪人类智慧巅峰与知识源头的神作。   

回到知识源头，108本好书囊括了「认知&神经&心理科学，网络科学，编程，数学，文艺创作」五大类21世纪的元学科，让你科学洗脑、独立自由、理性思维、开启心智不再发愁。
